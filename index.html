Team 1's Class Progression

Misty

MistyRobotics have created a safe first exposure to robotics. This safety first approach causes there to be a lot of limitations that can take a lot of effort to circumvent. This is furthered by no access to the code base.

  • Misty Blockly
  • Misty Python

Pupper

MangDang, creator of the Mini-Pupper, have packaged a group of robotics software into a quadruped capable of nearly lifelike movement. Though they are in constant development, the powerful tools and base systems allow for limitless creativity. As a drawback, the mini-pupper's have a lot of breaking points. Complex and interesting problems can arise from either the software or the hardware.

  • Pupper Assembly
  • Pupper Racing
  • End of Term Project: SLAM
